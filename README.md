# utilitybeltrstudio

Assorted Rstudio Addins

## Installation
 ```
 remotes::install_github("selkamand/utilitybeltrstudio")
 ```

## Usage

### Wide to long
#### Description
Convert: 
```
c("bob", "billy", "bonny")
```
To:
```
c(
"bob",
"billy",
"bonny"
)
```
#### How to Run
1. Highlight text
2. `[Ctrl/Command] + Shift + P`
3. Search for `wide_to_long`
